


[클래스 명명 접두사] - 여러 씬에서 자주  사용하는 것들(CameraMove)는 이름을 짓기 애매하기 때문에 씬마다 접두사를 정해서 사용한다.
유니티 에디터 상에서 네임스페이스가 적용이 안되기 때문에 다음과 같은 방법을 사용해야 한다.

Stage Select -> SS

InGame -> IG

사용 예) SS_CameraMove

[미니게임 개발시 중요 사항]
왠만하면 스테이지마다 파생 클래스를 만들지 말고, 게임 씬 내에서 프리팹이나 리소스 교체, 수치 조정으로 해결
확장에 염두를 두고 개발

[언어 처리]
다른 언어 추가의 확률이 있기 때문에 Json으로 언어를 관리, 동적으로 텍스트 처리
